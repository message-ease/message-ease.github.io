Privacy Policy

This policy explains how Message Ease handles information when you use our SMS and MMS messaging app.

This Privacy Policy has been created to disclose our information collection, use and disclosure practices for the Message Ease ("Message Ease") application.

Message Ease is an application on Android mobile phones that allows users to send and receive messages via their mobile carrier (using the SMS and MMS protocol). Messages are only ever transported between mobile phones and mobile carriers. We do not collect and do not store these messages on our servers. Also, we never share these messages with third parties.

We conform our data use to the European Union’s ("EU") General Data Protection Regulation ("GDPR"), with effect from 7 May 2020. 2. Event Data Upload

2.1 Why we upload event data

We collect event data to enhance the overall functionality and user experience of Message Ease. This data allows us to monitor feature usage and identify areas for enhancement, which helps us deliver a more efficient and effective communication tool to our users.

2.2 Specific SMS Event Data

Default SMS Success and Failure

We upload data on the success and failure of default SMS messages. This specific data helps us analyze and improve the reliability of our messaging services.

2.3 How we protect users' privacy

Protecting your privacy is of paramount importance to us. All event data collected is securely processed and stored. We have implemented robust security measures to ensure that no personally identifiable information (PII) is disclosed or misused. The data collected is anonymized and used solely for performance analysis and service improvements.

    7. Features That May Involve Your Privacy

    6.1 Number Location query

    Number Location is mainly used to query the attribution and carrier information of mobile phone numbers. Some advanced versions also support real-time location tracking and scam number identification. This feature is widely used for displaying caller ID location and managing contacts.

    To query Number Location information, the number you enter will be uploaded to our server for processing. During this process:

    • The number is transmitted encrypted for security.
    • It is temporarily stored for query execution.
    • Once the query is completed, the number is immediately deleted and never stored or shared.

    6.2 Recover password from Security Email

    The private message feature is designed to enhance the security of user messages and requires users to set a password. To prevent users from forgetting their passwords, Message Ease requires them to set up a private email to receive password verification codes when needed. The designated private email will serve as the secure contact for the account, ensuring that users can successfully recover their passwords and regain access to their private messages. It will be used solely for this purpose.

    We retrieve email addresses only for the following purposes:

    • Sending user feedback responses.
    • Resetting passwords if forgotten.
    Your email address is never shared or used for other purposes.

    6.3 Send HD MMS

    HD MMS is an enhanced version of MMS designed to address the issue of automatic compression and image quality degradation when sending pictures via traditional MMS. With HD MMS, users can send images, videos, and other multimedia content in higher resolution with less compression loss, thereby improving the visual experience.

    When sending HD MMS, the selected images are temporarily stored on our servers. These images are:

    • Deleted within 24 hours after the recipient views them.
    • Automatically removed after 7 days if not accessed.
    • Never leaked or shared with third parties.
